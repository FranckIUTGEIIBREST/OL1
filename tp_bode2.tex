
\section{Écriture d'une fonction en {\fontfamily{lmtt}\selectfont Python} et représentation graphique}
Pour définir une fonction qui n'est pas prédéfinie dans le logiciel (comme le sont les fonction $\cos$, $\sin$, $\tan$...), on utilise un procédé comparable à celui des fonctions en langage C.\\
Si on veut faire une représentation graphique d'une fonction $f$, il faut créer un vecteur échantillonage (abscisses) puis un vecteur image de ces abscisses.
\bigskip\\
\textit{Exemple} :\\ La syntaxe du tracé de la fonction $f$ définie par $f(t)=\displaystyle \frac{1}{1+t^2}$ sur l'intervalle $[-3;5]$ pourrait être la suivante:\\
Attention! C'est un exemple! Ce n'est pas la fonction qui vous sera demandée. Prenez le temps de bien comprendre la syntaxe.

from matplotlib.pyplot import *

def f(t):
   import numpy as np  # non necessaire car deja importee
   import matplotlib.pyplot as plt  # non necessaire car deja importee
   return 1/(1+t**2)  # 1/(1+t^2)

\section{Fonction logarithmique}
\begin{Def}[Logarithme de base $a$]
Soit $a$ un réel strictement positif, différent de 1. On définit la fonction logarithme de base $a$ par $$\log_a (x)=\frac{\ln x}{\ln a}$$
\end{Def}

\section{Gain}
\begin{Def}[Gain]
Les électroniciens utilisent le \textbf{gain} défini par $$\displaystyle G(\omega)=20\log_{10}\left( \left| H(\omega) \right| \right)$$

Pour la représentation graphique, l'échelle des abscisses est logarithmique $i.e$ on représente, sur l'axe des abscisses, les valeurs de $\log_{10} (\omega)$ et non les valeurs de $\omega$.
\end{Def}
