\begin{center}
\Large{Syllabus}
\end{center}
Le programme pédagogique national du diplôme universitaire de technologie spécialité Génie Électrique et Informatique Industrielle comprend, en première année, deux modules intitulés "Outils logiciels". 
Les premiers Travaux Pratiques vont vous permettre de vous familiariser avec le langage Python, en utilisant l'environnement de développement intégré \texttt{Spyder}. Vous pouvez télécharger gratuitement le logiciel \texttt{Spyder} à l'adresse
\begin{center}
\textbf{http://pythonhosted.org/spyder/installation.html}
\end{center}
\bigskip
Les thèmes abordés durant l'année seront :
\begin{enumerate}
\item Les représentations graphiques
\item Les fonctions sinusoïdales
\item Les diagrammes de Bode
\item Les asymptotes
\item Les polynômes
\item La décomposition en éléments simples
\item L'intégration
\item Les séries de Fourier
\end{enumerate}
